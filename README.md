Генератор Профиля Волнового Редуктора с Промежуточными Телами Качения (ВПТК)
----
Данная программа предназначена для быстрого расчёта и построения профиля волновой передачи с телами качения (ВПТК). 

ВПТК - механическая передача, имеющая большое передаточное число (даже больше 20 в одной ступени). Редуктор с такой передачей является аналогом циклоидного или волнового редуктора (без промежуточных тел качения).
Также его можно распечать на 3D-принтере, в нём нет каких-то дорогих деталей, что делает его одним из лучших редукторов для домашней робототехники. 

Однако его профиль имеет сложную геометрию, и чтобы его смоделировать удобно использовать данный генератор профиля. После автоматического расчёта и построения по заданным параметрам, можно скачать профль в формате dxf и открыть в любой CAD-программе. 

За основу проекта был взят скрипт Артёма TrashRobotics: https://codeberg.org/TrashRobotics/BallsWaveGearingGenerator

Я оформил его в виде сайта для удобства использования. Теперь не нужно устанавливать Python и сам код на ваш компьютер. Достаточно открыть страницу в любом браузере и через удобный интерфес пользоваться генератором.

Как пользоваться
----
1. Открыть главную станицу сайта: https://marsovich20.pythonanywhere.com
2. Ввести необходимые параметры в поля ввода (при наведении курсора на некоторые надписи, появляется всплывающая подсказка)
3. При необходимости можно поменять настройки генерации профиля
4. Нажать кнопку "Рассчитать"
5. Если все значения введены верно, то прогамма успешно сгенерировала профиль, и теперь можно скачать два файла: чертёж (в формате dxf) и параметры передачи (в формате txt).

Если какие-то параметры будут введены неверно, программа на это укажет.

Файлы
----
main.py - главный файл, является точкой входа. На локальном сервере запускайте именно его. 

calculator.py - отвечает исключительно за расчёты, вычисления и создание файлов. Может использоваться самостоятельно или в других проектах.

texts.py - содержит словари со всеми текстами на сайте для русской и английской версии.

Остальные файлы имеют стандартное назначение и не требуют дополнительных комментариев.

Библиотеки
-----
Для работы кода на компьютере (из IDE) требуются библиотеки ezdxf, numpy и фреймворк flask. Установить их можно через pip.
